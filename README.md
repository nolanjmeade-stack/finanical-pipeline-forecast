# Financial-pipeline-forecast
End-to-end crypto data pipeline using Python, Docker, PostgreSQL, and Prophet. Fetches real-time data, stores it in a database, and runs 30-day Bitcoin forecasts. Designed for solo deployment with optional Power BI integration for analytics and dashboards.
